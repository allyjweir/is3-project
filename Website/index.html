IS3 Team Project

As part of the Information Systems 3 course at the University of Glasgow, we were tasked with creating a useful and informative data visualisation combining information of medal wins from the London 2012 Olympics and data collected by the World Health Organisation. This was successfully created using d3, jQuery, Bootstrap and the necessary Javascript we cobbled together.
